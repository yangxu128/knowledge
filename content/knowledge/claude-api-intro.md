## Claude 简介

Claude 是由 Anthropic 构建的高性能、可信赖且智能的 AI 平台。Claude 擅长处理涉及语言、推理、分析、编程等任务。

### 最新一代 Claude 模型

- **Claude Opus 4.7** - Anthropic 最强大的模型，适用于复杂推理和智能体编程，相比 Claude Opus 4.6 有质的飞跃。
- **Claude Sonnet 4.6** - 前沿智能的大规模应用，专为编程、智能体和企业工作流而构建。
- **Claude Haiku 4.5** - 最快的模型，具备接近前沿的智能。

### 两种使用方式

| | Messages API | Claude 托管智能体 |
|---|---|---|
| **是什么** | 直接的模型提示访问 | 预构建、可配置的智能体框架，在托管基础设施中运行 |
| **最适合** | 自定义智能体循环和细粒度控制 | 长时间运行的任务和异步工作 |

### 新手推荐路径

1. **进行第一次 API 调用** - 设置环境，安装 SDK，并向 Claude 发送第一条消息。
2. **了解 Messages API** - 学习核心请求和响应结构，包括多轮对话、系统提示和停止原因。
3. **选择合适的模型** - 按能力和成本比较 Claude 模型，选择最适合您用例的模型。
4. **探索功能和工具** - 发现 Claude 的能力：扩展思维、网络搜索、文件处理、结构化输出等。

### 核心能力

- **文本和代码生成** - 文本摘要、问题回答、数据提取、文本翻译以及代码解释和生成。
- **视觉** - 处理和分析视觉输入，从图像生成文本和代码。

### 开发者工具

- **开发者控制台** - 在浏览器中使用 Workbench 和提示生成器进行原型设计和测试。
- **API 参考** - 探索完整的 Claude API 和客户端 SDK 文档。
- **Claude Cookbook** - 通过交互式 Jupyter 笔记本学习 PDF、嵌入等内容。
